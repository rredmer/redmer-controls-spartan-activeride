## Introduction

ActiveRide is an application developed by Redmer Controls for tuning the active suspension systems of Spartan Motors vehicles. The application provides a user-friendly interface for configuring and calibrating various parameters of the suspension system to ensure optimal performance and comfort.

## Features

- Parameter configuration and tuning
- Firmware download and validation
- Diagnostic code management
- User management
- Logging and error handling

## Installation

### Prerequisites

- Windows operating system
- Microsoft Visual Studio
- Required DLLs and OCX files (listed in `ActiveRide.PDM`)

### Steps

1. Clone the repository:
    ```sh
    git clone https://github.com/your-repo/redmer-controls-spartan-activeride.git
    ```

2. Open the project in Microsoft Visual Studio.

3. Build the project to generate the executable files.

4. Run the setup executable located in `ActiveRide Setup/setup.exe` to install the application.

## Usage

1. Launch the ActiveRide application from the Start menu or the installation directory.

2. Follow the on-screen instructions to configure and calibrate the suspension system.

3. Use the various forms and tools provided in the application to manage parameters, download firmware, and handle diagnostics.

## Project Structure

### Key Files

- [ActiveRide.csi](http://_vscodecontentref_/0): Contains the history of changes and modifications.
- [ActiveRide.PDM](http://_vscodecontentref_/1): Project deployment manager file listing dependencies and setup configurations.
- [main.bas](http://_vscodecontentref_/2): Main application procedure and form loading.
- `ActiveRide Setup/setup.exe`: Setup executable for installing the application.
